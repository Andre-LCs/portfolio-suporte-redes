# 🛠️ Problema: Assinador não funciona (NET Framework desativado)

## 📌 Solução

Acesse painel de controle > programas > programas e recursos > ativar ou desativar recursos do Windows.

Uma vez aberta a caixa, abra e marque a pasta do NET Framework (ativação HTTP e ativação NÃO HTTP).

Marque as subpastas do NET Framework (são apenas duas = "ativação HTTP..." e "ativação NÃO HTTP...").

Clique em "OK".

O Windows baixará os arquivos necessários (aparece uma barra verde indicando a instalação).

Pronto.

O assinador livre volta a funcionar normalmente.
